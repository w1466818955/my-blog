# 1.a 标签-基于文件格式使用不同的样式

/* 链接 */
a[href^="http://"]
{
    padding-right: 20px;
    background: url(external.gif) no-repeat center right;
}

/* emails */
a[href^='mailto:'] {
    padding-right: 20px;
    background: url(email.png) no-repeat center right;
}

/* pdfs */
a[href$='.pdf'] {
    padding-right: 20px;
    background: url(pdf.png) no-repeat center right;
}

# 2.页面滚动条

  • 隐藏滚动条
::-webkit-scrollbar {
    display: none;
}
1
2
3
  • 修改滚动条样式
/* 边角，即两个滚动条的交汇处 --> 粉色(只能设置颜色。大小可跟随轨道变化) */
::-webkit-scrollbar-corner {
    background: pink;
}

/* 滚动条整体部分 --> 红色(可以设置颜色、大小) */
::-webkit-scrollbar {
    width: 10px;
    height: 10px;
    background: red;
}

/* 滚动条的轨道 --> 红色(只能设置颜色) */
::-webkit-scrollbar-track {
    background: #bf2709;
}

/* 内层轨道 --> 黄色(只能设置颜色) */
::-webkit-scrollbar-track-piece {
    background: #fff;
}

/* 滚动条里面的小方块 --> 蓝色(可以设置颜色) */
::-webkit-scrollbar-thumb {
    border-radius: 10px;
    background: #1499d6;
}

/* 滚动条的轨道的两端按钮 --> 黑色(可以设置颜色、大小) */
::-webkit-scrollbar-button {
    width: 10px;
    height: 10px;
    background: #666666;
}

# 3.包裹长文本

当文本的长度超出容器自身宽度时，可以通过设置 white-space: pre-line; word-wrap: break-word;将长文本包裹起来

.pre {
    white-space: pre-line;
    word-wrap: break-word;
}
